# AI Risk Intelligence System 

This project uses Artificial Intelligence and NLP to detect behavioural red flags, emotional manipulation, and safety risks in conversations.

## Features
- Secret behaviour detection
- Emotional manipulation detection
- Urgency and pressure detection
- Toxicity analysis
- Emotion detection
- Risk scoring engine

## Tech Stack
Python, NLP, Machine Learning, Deep Learning, Flask

## Goal
To build a smart safety and behavioural analysis platform.
